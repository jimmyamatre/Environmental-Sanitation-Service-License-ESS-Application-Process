The ESS License Application Process was developed to digitize and streamline the licensing of environmental sanitation service providers operating within Kampala. It ensures transparent, efficient, and compliant management of waste collection, transportation, and disposal services.

Online Application: Service providers submit applications and required documentation digitally.
